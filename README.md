# Nexus Coworking Space Manager

Project is reorganized into separated frontend and backend layers.

## Project Structure

```
my-app-frontend/
my-app-backend/
```

## Frontend

- Location: `my-app-frontend`
- Stack: React Router v7, React Bootstrap, Vite

Run:

```bash
cd my-app-frontend
npm install
npm run dev
```

## Backend

- Location: `my-app-backend`
- Status: clean architecture scaffold for API layer

Run:

```bash
cd my-app-backend
npm install
npm run start
```
